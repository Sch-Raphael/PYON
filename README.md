# PYON BETA
PYON (Python Online) is a fully featured Browser made with PyQt5.

### features
- Tor Routing
- Custom Proxy
- Youtube AD block
- AD, Tracker, Cookie blocker
- disable javascript loading
- User Agent changer (Random, Custom)
